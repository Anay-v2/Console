VS Code

VS Code (or Visual Studio Code) is an IDE(integrated development environment) that provides you a code editor, syntax hylighting, and more. It also has some extensions you can download. To instal VS Code go Here and click the download button.

Console

The console is a tool that you can use to test your code. To open the console, right click and click inspect and then go to the console. Console allows you to see if certain things are working in your code. In the console you can type any javascript you want. But the more preffered way is to use console.log("Console");. You can also do things like console.error("this is an error.");. Check the console to see some things I have done using it.
